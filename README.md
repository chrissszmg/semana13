bloque0_warmup:

Contiene a los ejercicios de diagnóstico donde se analiza la diferencia entre pasar parámetros por valor y por referencia.

bloque1_referencias:

Incluye ejercicios para entender cómo funcionan las referencias dentro del codigo, 
cómo actúan como alias de una variable y cómo permiten modificar valores originales.

bloque2_punteros:

Ejercicios relacionados con punteros y prediccion de la salida (output) del codigo, direcciones de memoria y desreferenciación usando los operadores * y &

bloque3_desafio:
  Ejercicio practico aplicando todo lo mencionado anteriormente el cual es
  un pequeño sistema de registro de notas de estudiantes usando arreglos, punteros y referencias para ajustar notas y calcular el promedio del curso.
